# Anonymous Feedback System

## 📌 Description
This project is a web-based Anonymous Feedback System that allows users to submit feedback without revealing their identity. It ensures privacy and encourages honest opinions.

## ⚙️ Technologies Used
- PHP
- MySQL
- HTML/CSS
- JavaScript
- XAMPP

## 📋 Features
- User Registration & Login
- Anonymous Feedback Submission
- View Feedback
- Dashboard Interface
- Secure Database Connection

## 🧰 Prerequisites
- XAMPP (Apache + MySQL)
- Web Browser
- PHP installed

## 🚀 Installation Steps

1. Download or clone the repository:
   git clone https://github.com/your-username/anonymous-feedback-system

2. Move the project folder to:
   C:\xampp\htdocs\

3. Start XAMPP:
   - Start Apache
   - Start MySQL

4. Open phpMyAdmin:
   http://localhost/phpmyadmin

5. Create a database (e.g., feedback_db)

6. Import the SQL file (if available)

7. Update database connection in:
   config/db.php

8. Run the project:
   http://localhost/anonymous-feedback-system
